## Задание

```bash
Требуется написать прототип API каталога товаров.
Поля товара: id, name.

Требуемые методы в API:

Публичные:
– Список товаров, с постраничным разбиением (pagination)
– Получение товара по id (добавить кэширование ответа на 1 минуту)
– Авторизация по почте и паролю. Возвращает JWT

Приватные (с авторизация по JWT):
– Добавление товара
– Редактирование товара по id

Обязательные компоненты:
– NestJS
– TypeOrm
– NestJS Crud (https://github.com/nestjsx/crud) предпочтительно
– Swagger (документация методов API)
остальное на усмотрение кандидата.

Желательно, но не обязательно:
– СУБД   MySQL
– Структуру БД “завернуть” в миграцию
– Запустить инфраструктуру проекта через docker-compose
– README
```
